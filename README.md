# $BROKE — The Broke Bull

Animated meme coin landing page built around the **Still broke. Still bullish.** narrative.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Before launch

Update these placeholders in `index.html` and `public/site.html`:

- official contract address;
- X, Telegram and Dexscreener links;
- buy link;
- liquidity and verification proof links.

Never publish unverified liquidity, holder or audit claims.
